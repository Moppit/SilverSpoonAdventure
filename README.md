## Getting Started

### Install Dependencies
```
pip3 install playsound
pip3 install Pillow
```

### Run Game
```
python3 main.py
```

### Helpful Information
- If you get stuck on a puzzle, you can hit q to leave that puzzle and try something else
- Stat Checks are calculated by adding your stat to the d20 that you roll
